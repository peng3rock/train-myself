# Train Myself

一个帮助你记录和追踪练习进度的应用。

## 功能特点

- **两种目标类型**：
  - **数值型目标**：设定一个目标数值（如跑100公里），每天记录练习数值，累计达到目标
  - **任务型目标**：设定一个目标分数（如测验达到100分），周期性进行测验并记录分数

- **进度追踪**：
  - 数值型目标显示累计进度条和完成百分比
  - 任务型目标显示测验次数和最近分数

- **数据持久化**：所有数据保存在浏览器本地存储中，无需担心数据丢失

- **美观界面**：现代化的渐变设计，响应式布局

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动

### 构建生产版本

```bash
npm run build
```

## 使用说明

1. **添加目标**：点击右上角"添加目标"按钮
   - 输入目标名称（如"每天跑步"）
   - 选择目标类型（数值型或任务型）
   - 设定目标值（数值或分数）

2. **记录练习**：
   - 点击目标卡片上的"添加记录"按钮
   - 输入今日的练习数值或测验分数
   - 可选添加备注

3. **查看进度**：
   - 数值型目标会显示累计进度条
   - 任务型目标会显示测验次数和最近分数
   - 完成的目标会显示"已完成"标识

4. **删除目标**：点击目标卡片右上角的"×"按钮

## 技术栈

- Vue 3
- Vite
- 本地存储（localStorage）

## 局域网访问

应用已配置为支持局域网访问，方便在手机上测试：

1. 启动开发服务器后，Vite会显示本地和网络地址，例如：
   ```
   Local:   http://localhost:5173/
   Network: http://192.168.1.100:5173/
   ```

2. 在手机上连接到同一WiFi网络

3. 在手机浏览器中输入Network地址（如 `http://192.168.1.100:5173/`）

4. 即可在手机上访问应用进行测试

**注意**：确保电脑和手机在同一局域网内，并且防火墙允许5173端口的访问。
